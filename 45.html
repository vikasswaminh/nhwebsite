CCNA Course Schedule

Our CCNA course at Networkers Home is designed to cover all the crucial topics required to pass the CCNA certification exam while ensuring students gain hands-on experience. Below is the detailed course schedule outlining the modules covered over a span of six weeks. This schedule is applicable for both in-class and CCNA course online students.

  • Week 1: Introduction to Networking and OSI Model – Understand the fundamentals of networking, the OSI model, and the TCP/IP protocol suite. Learn how data is transferred across networks using IP addressing.
  • Week 2: IP Addressing, Subnetting, and IPv4/IPv6 Basics – Master the concepts of IP addressing, subnetting, and the differences between IPv4 and IPv6. Practice configuring network devices for efficient data routing.
  • Week 3: Routing Concepts and Protocols – Gain hands-on experience with routing protocols like RIP, OSPF, and EIGRP. Learn how to configure routers for inter-network communication.
  • Week 4: VLANs, STP, and EtherChannel – Understand the importance of Virtual LANs (VLANs) in network segmentation, configure Spanning Tree Protocol (STP) for network redundancy, and deploy EtherChannel for bandwidth aggregation.
  • Week 5: Network Security, Firewalls, and VPNs – Explore network security basics, including firewalls, VPNs, and access control. Learn how to secure network devices and prevent unauthorized access.
  • Week 6: Capstone Project and CCNA Exam Preparation – Apply all the concepts learned in a real-world capstone project and prepare for the CCNA certification exam with practice tests and simulations.

Each week includes lab sessions where students practice configuring real devices and work on troubleshooting scenarios. By the end of the course, you will be fully prepared to take and pass the CCNA exam.

CCNA Course Curriculum

Explore the comprehensive curriculum designed to provide hands-on experience and a deep understanding of networking concepts, technologies, and tools.

In this module, you’ll learn the basic concepts of networking, which form the foundation for understanding more advanced topics. You’ll cover the OSI and TCP/IP models, network topologies, and key components like routers, switches, and firewalls.

  • OSI Model & TCP/IP Model: Detailed explanation of the 7 layers in the OSI model and the 4 layers in the TCP/IP model, and how data flows through each layer.
  • Network Topologies: Bus, Star, Ring, and Mesh topologies and how they influence network design and performance.
  • Cisco Routers and Switches: Understanding routers, switches, hubs, and bridges, and their roles in network communication.
  • IPv4 and IPv6 Addressing: Basics of IP addressing, subnetting, and network masks. Introduction to IPv6 addressing schemes.
  • Data Encapsulation: How data packets are created, encapsulated, and transmitted across a network.
  • Network Devices: Overview of key devices like routers, switches, modems, firewalls, and access points.
  • Physical Media: Introduction to cables (Ethernet, Fiber Optic, Coaxial), and wireless communication standards (Wi-Fi).

This module dives deep into configuring, managing, and troubleshooting routers and switches. You will gain hands-on experience with Cisco hardware and software.

  • Static and Dynamic Routing: Understanding static routing, dynamic routing protocols like RIP, OSPF, and EIGRP.
  • Routing Tables: Learning how routers make forwarding decisions based on routing tables.
  • VLANs and Trunking: Configuring VLANs and how trunking is used to manage multiple VLANs across switches.
  • Spanning Tree Protocol (STP): Understanding how STP prevents loops in a network and ensures redundancy.
  • Switch Configuration: Configuring Cisco switches for security, VLANs, and performance optimization.
  • Router Configuration: Hands-on practice configuring Cisco routers using the CLI interface for IP routing, static routes, and NAT.
  • EtherChannel: Configuring EtherChannel for link aggregation to increase bandwidth and redundancy.

In this module, you will learn the various IP services that are essential for modern networking, including DHCP, DNS, NAT, and NTP.

  • DHCP (Dynamic Host Configuration Protocol): Configuring and troubleshooting DHCP servers and how they assign IP addresses to clients.
  • DNS (Domain Name System): Understanding how DNS works, configuring DNS servers, and troubleshooting domain name resolution issues.
  • NAT (Network Address Translation): Implementing NAT to translate private IP addresses to public IPs, enabling internet connectivity for internal devices.
  • Access Control Lists (ACLs): Configuring ACLs for traffic filtering and security on Cisco routers and switches.
  • WAN Connectivity: Introduction to WAN technologies like MPLS, Metro Ethernet, and broadband connections.
  • NTP (Network Time Protocol): Configuring NTP for time synchronization across network devices.
  • IP SLA (Service Level Agreement): Implementing IP SLA to measure network performance and ensure QoS (Quality of Service).

Learn about the essential security concepts needed to protect modern networks, including firewalls, VPNs, access control, and encryption.

  • Firewalls: Configuring firewall rules and understanding how firewalls filter network traffic for enhanced security.
  • VPNs (Virtual Private Networks): Implementing VPNs for secure remote access and site-to-site connectivity.
  • Encryption: Learning about encryption protocols like SSL, TLS, and IPsec for secure data transmission.
  • Network Authentication: Configuring network authentication using protocols like 802.1x and RADIUS.
  • Access Control: Understanding MAC address filtering, port security, and network-based access control (NAC).
  • Security Best Practices: Applying security policies and procedures to mitigate risks like DDoS attacks, malware, and phishing.
  • VPN Configuration: Configuring SSL VPN and IPsec VPN for secure remote access to corporate networks.

Network automation is becoming increasingly important in modern network environments. In this module, you'll learn the basics of network automation and how it can enhance efficiency.

  • Introduction to Automation: Understanding the need for automation in networking and the basic concepts behind programmable networks.
  • Python for Network Engineers: Learning Python scripting for automating common network tasks.
  • REST APIs: Using REST APIs to interact with network devices and automate configuration management.
  • SDN (Software-Defined Networking): Introduction to SDN architecture and how it enables centralized network management.
  • Network Programmability: Using tools like Ansible and Puppet to automate network configuration.
  • Model-Driven Telemetry: Gathering network performance data in real-time through telemetry for proactive monitoring.
  • Automation Tools: Introduction to network automation tools like Cisco DNA Center and Python scripts for configuration tasks.

This module focuses on the best practices for preparing and passing the CCNA certification exam. You'll go through mock tests, sample questions, and a review of key concepts.

  • CCNA Exam Overview: Understanding the structure of the CCNA exam, topics covered, and how the questions are distributed.
  • Exam Strategies: Tips and strategies for managing time, solving multiple-choice questions, and answering scenario-based questions effectively.
  • Mock Tests: Practice with CCNA mock exams to get familiar with the test format and timing.
  • Review of Key Concepts: A comprehensive review of essential concepts like routing, switching, IP addressing, network security, and automation.
  • Study Resources: Recommended study materials, practice labs, and simulators to enhance your preparation.
  • Exam Simulation: Experience a real exam environment with CCNA exam simulations.
  • Last-Minute Tips: Guidance on how to revise efficiently in the final days leading to the exam.
